\chapter{总结与展望}
\section{总结}


互连网络作为高性能计算的重要组成部分，其容错性能的好坏对网络的运行至关重要。
扩展立方体作为超立方体的一种变体，拥有许多优秀的性质。我们在本文中重点研究了扩展立方体的容错性能，其中包括结构连通度、子结构连通度和额外诊断度。取得的具体成果如下：


一、扩展立方体的结构连通度及子结构连通度

我们求出了$n$-维扩展立方体$AQ_n$的$H$-结构连通度$\kappa(AQ_n,H)$和$H$-子结构连通度
$\kappa^s(AQ_n,H)$，其中$H \in \{K_{1,M}，P_L,C_N\}$，
且$1 \leq M \leq 6, 1 \leq L \leq 2n-1, 3 \leq N \leq 2n-1$。
研究成果如下：

(1) $\kappa(AQ_n,K_{1,M})=\kappa^s(AQ_n,K_{1,M})=\left\{
\begin{array}{rcl}
2n-1       &      &\textnormal{其中}n \geq 4 \textnormal{ , } K_{1,M}=K_1,\\
\lceil\frac{2n-1}{1+M}\rceil     &      &\textnormal{其中}n \geq 4 \textnormal{ , } 1 \leq M \leq 3,\\
\lceil\frac{n-1}{2}\rceil     &      &\textnormal{其中} n \geq 6 \textnormal{ , } 4 \leq M \leq 6.
\end{array} \right. $\\


(2) $
\kappa(AQ_n,P_L)=\kappa^s(AQ_n,P_L)=\lceil\frac{2n-1}{L}\rceil \quad \quad\textnormal{其中 } n \geq 3 \textnormal{ , } 1 \leq L \leq 2n-1.\\
$

(3) $
\kappa^s(AQ_n,C_N)=\lceil\frac{2n-1}{N}\rceil \quad \quad\textnormal{其中} n \geq 3 \textnormal{ , } 3 \leq N \leq 2n-1.\\
$

(4) $ \kappa(AQ_n,C_N)=\left\{
\begin{array}{rcl}
n-1     &      &\textnormal{其中} n \geq 6 \textnormal{ , } N=3,\\
\lceil\frac{2n-1}{N-1}\rceil       &      &\textnormal{其中} n \geq 6 \textnormal{ , } 4 \leq N \leq 2n-1.
\end{array} \right. $\\

同时， 我们给出了扩展立方体在小于$K_{1,6}$-子结构连通度下的容错路由算法AQFRouting，该算法的时间复杂度为$O(\lceil \textrm{log}_2|F| \rceil n^3)$，且该算法的所构造的最大无故障路径长度为$8m+  \lceil\frac{n-m}{2} \rceil  + 1$，其中$m=\lceil \textrm{log}_2|F| \rceil$，$F$为故障顶点集合，且我们通过模拟实验验证了算法的有效性。

二、扩展立方体的额外诊断度和诊断算法

基于额外诊断度的定义，
我们求出了$n$-维扩展立方$AQ_n$的$2$-额外诊断度为$6n-15$，
其中$n \geq 8$，并给出了其相关证明。
同时， 我们给出了扩展立方体在$MM^*$模型下的$2$-额外诊断算法
AQ-EXTRA-DIAG，该算法的复杂度为$O(N^2)$，其中$N$为扩展立方体的顶点个数。
且我们通过模拟实验验证了算法的有效性。



\section{展望}

本文在扩展立方体的结构连通度和诊断度研究上取得了一些创新性进展。
但是总结下来，我们的结果仍有许多不足之处，希望今后作出进一步的成果，
主要包括以下几方面：

1. 进一步将故障结构扩展到更具通用性的结构上面，而不是仅仅局限于一些小的特殊的故障结构，同时希望将结论一般化，使其可以应用到更多的互连网络拓扑结构上面。

2. 借助于扩展立方体的结构连通度及子结构连通度研究，希望将其应用于扩展立方体的其他结构容错性质，比如哈密顿性质等\upcite{LvY2018,LvY2017}。

3. 进一步探究扩展立方体的$g$-额外诊断度，希望可以将$g$一般化，而不是局限于某些特定的整数值。
